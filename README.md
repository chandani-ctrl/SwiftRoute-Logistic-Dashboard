🚚 SwiftRoute Logistic Dashboard
📌 Project Overview

The SwiftRoute Logistic Dashboard is an interactive logistics performance dashboard designed to monitor and analyze order fulfillment, delivery efficiency, driver performance, hub capacity, and vehicle utilization.

This dashboard provides a single view of logistics operations for management to track KPIs, identify delays, and optimize routes, drivers, and fleet usage.

📅 Reporting Period: February 2024

🎯 Objectives

Track overall logistics performance

Monitor on-time delivery rate

Analyze hub capacity vs orders

Evaluate driver experience and delays

Monitor vehicle availability and usage

Improve customer satisfaction (CSAT)

📊 Key KPIs

Total Orders: 1,130

On-Time Delivery Rate: 81.63%

Customer Satisfaction (CSAT): 86.5%

Average Delivery Time: 35.94 hours

📈 Month-over-Month comparison included for all KPIs.
